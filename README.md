# Mobile Gesture Carousel Demo

Una demo interattiva di un carosello controllato tramite gesture circolari su dispositivi mobili.

## Caratteristiche

- **Gesture circolari**: Controlla il carosello con movimenti circolari nella parte inferiore dello schermo
- **Pull-to-refresh**: Tira verso il basso dalla parte superiore per ricaricare la pagina
- **Ottimizzato per mobile**: Interfaccia completamente ottimizzata per dispositivi mobili
- **Blocco swipe nativi**: Previene gli swipe laterali del browser per una migliore esperienza utente

## Come funziona

- **Scroll del carosello**: Esegui un movimento circolare orario nella barra inferiore per scorrere avanti, antiorario per tornare indietro
- **Pull-to-refresh**: Tira verso il basso dalla parte superiore dello schermo per ricaricare la pagina
- **Debug mode**: Fai doppio tap sull'area di gesto per attivare/disattivare le informazioni di debug

## Tecnologie utilizzate

- HTML5
- CSS3 (con supporto per viewport dinamico mobile)
- JavaScript vanilla (senza dipendenze esterne)

## Setup locale

1. Clona il repository:
```bash
git clone https://github.com/TUO_USERNAME/app-incontri.git
cd app-incontri
```

2. Apri `index.html` nel tuo browser mobile o usa un emulatore mobile.

## Deploy su GitHub Pages

Il progetto è configurato per funzionare automaticamente con GitHub Pages. Dopo aver fatto push del codice:

1. Vai su Settings del tuo repository GitHub
2. Scorri fino a "Pages" nella sidebar
3. Seleziona il branch `main` (o `master`) come source
4. Clicca "Save"
5. Il sito sarà disponibile su: `https://TUO_USERNAME.github.io/app-incontri/`

## Browser supportati

- Safari iOS (consigliato)
- Chrome Mobile
- Firefox Mobile
- Altri browser moderni con supporto per touch events

## Note

Questa pagina è ottimizzata esclusivamente per dispositivi mobili. Su desktop verrà mostrato un messaggio informativo.
